# 网盘搜索APP编译说明

## 项目信息

这是一个全新的网盘搜索Android应用，功能类似混合盘，但更简洁。

**主要功能：**
- 支持搜索网盘资源（使用 pansou.com 搜索引擎）
- 简洁的界面设计
- WebView显示搜索结果

## 编译方法

### 方法1：使用GitHub Actions（推荐）

1. 将此项目推送到GitHub仓库
2. 在GitHub上创建一个新的仓库
3. 将项目代码推送到该仓库
4. GitHub Actions会自动编译APK
5. 从Actions页面下载编译好的APK

具体步骤：

```bash
# 在GitHub上创建新仓库后
cd ~/storage/downloads/pansou-app
git remote add origin https://github.com/你的用户名/pansou-app.git
git branch -M main
git push -u origin main
```

然后访问GitHub仓库的Actions页面，等待编译完成，下载APK。

### 方法2：本地编译（需要安装Java和Android SDK）

1. 安装Java JDK 17：
```bash
pkg install openjdk-17
```

2. 下载并安装Android SDK：
```bash
pkg install android-sdk
# 或者从 https://developer.android.com/studio 下载Android Studio
```

3. 配置环境变量：
```bash
export ANDROID_HOME=/path/to/android-sdk
export PATH=$PATH:$ANDROID_HOME/tools:$ANDROID_HOME/platform-tools
```

4. 接受Android SDK许可：
```bash
yes | sdkmanager --licenses
```

5. 安装必要的SDK组件：
```bash
sdkmanager "platform-tools" "platforms;android-34" "build-tools;34.0.0"
```

6. 编译APK：
```bash
cd ~/storage/downloads/pansou-app
./gradlew assembleRelease
```

7. APK位置：
```
app/build/outputs/apk/release/app-release-unsigned.apk
```

### 方法3：使用在线APK构建服务

1. 访问 https://build.phonegap.com/
2. 注册账号
3. 上传项目文件
4. 在线编译APK

### 方法4：使用GitHub Codespaces

1. 访问 https://github.com/
2. 创建新仓库并上传此项目
3. 使用GitHub Codespaces打开项目
4. 在Codespaces中编译APK
5. 下载编译好的APK

## 使用说明

1. 安装APK后打开应用
2. 在输入框中输入搜索关键词
3. 点击"搜索"按钮
4. 在WebView中查看搜索结果

## 修改搜索源

如果要修改搜索源，编辑 MainActivity.java 文件：

```java
String url = "https://pansou.com/s/" + query;
// 修改为其他搜索源
String url = "https://other-search-site.com/search?q=" + query;
```

## 注意事项

1. 首次编译需要下载Gradle和Android SDK，可能需要较长时间
2. 编译过程需要稳定的网络连接
3. 编译后的APK需要签名才能发布到应用商店
4. 建议使用方法1（GitHub Actions）编译，最简单快捷
